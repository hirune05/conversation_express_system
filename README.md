# 感情表現データ収集システム

## セットアップ

### ローカルLLM:
1. Ollamaがインストールされていない場合は、お使いのPCにOllamaをインストールしてください。

```bash
brew install ollama
```

2. Qwen3:8bがインストールされていない場合はインストールしてください。
```bash
ollama pull qwen3:8b
```

3. Ollamaを起動し、起動済みであることを確認してください
```bash
ollama serve
```

### サーバ:

1. 各ライブラリがインストールされていない場合はインストールしてください。

2. サーバーを起動:
```bash
python app.py
```

3. ブラウザで開く:

[http://localhost:3000](http://127.0.0.1:5000)

## 使い方

チャット欄で会話をすると会話に応じてキャラクタの表情が変わります。
